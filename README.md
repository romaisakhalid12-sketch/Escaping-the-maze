# Escaping-the-maze

A Python solution that helps Reeborg successfully navigate through a maze using logical decision-making and loops.

## 🎯 Project Overview

The program guides Reeborg from the starting position to the goal by following a simple maze-solving algorithm. It continuously checks the surroundings and decides whether to move forward, turn right, or turn left until the destination is reached.

## ✨ Features

- 🧭 Automatically solves different maze layouts
- 🔄 Uses loops for continuous navigation
- 🤖 Makes decisions based on the path ahead
- 🐍 Demonstrates Python functions and conditional statements

## 💻 Platform

This project was created using **Reeborg's World**, an online platform for learning Python programming through interactive robot challenges.
